# ryan-leung.com

## Stack

1. Next.js
1. TailwindCSS
1. Playwright

## Getting Started

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Tests

Install playwright
```bash
npx playwright install
```
